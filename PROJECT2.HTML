SECUREFILEX: A COMPREHENSIVE FILE SECURITY APPLICATION

Place of Work: B. S. Abdur Rahman Crescent Institute of Science and Technology, Chennai

Abstract: SecureFileX is a comprehensive file security application designed to protect sensitive data using AES-256 encryption, multi-factor authentication (MFA), and role-based access control (RBAC) to prevent unauthorized access and cyber threats. It ensures secure file storage, transfer, and integrity verification through SHA-256 hashing, while incorporating real-time monitoring, intrusion detection, and audit logging for enhanced security. Built using Python (Flask/Django) for backend processing and React.js for an intuitive frontend, SecureFileX provides a user-friendly, cross-platform solution suitable for individuals and enterprises. With zero-trust architecture and machine learning-based anomaly detection, it proactively safeguards files against cyber threats, ensuring confidentiality, integrity, and availability in cloud and local environments.

Technologies Used:HTML, CSS, JAVASCRIPT, PYTHON

libraries Used:FLASK LIBRARY, AES 256 Algorithm
